자바스크립트의 객체 시스템

자바스크립트는 브라우저 자체를 객체로 이해하고 있으며 최상위 객체를 window객체라고 부른다.

그리고 그 최상위 객체 window아래에는 여러 객체들이 포함되어 있다.

  • Document -> 정보들을 표현하는 문서객체
    • // HTML, XML 등의 문서의 정보를 담고 있다.
  • location -> 웹페이지의 위치 정보를 표현하는 객체
    • // URL, 프로토콜, 호스트, 포트, 경로 등의 정보를 담고 있다.
  • history -> 이동했던 페이지들에 대한 정보를 관리하는 객체
    • // 방문했던 페이지들의 URL, 방문 순서, 방문 횟수 등의 정보를 담고 있다.
  • console -> 개발자 툴에 있는 콘솔 객체
    • // 로그, 경고, 오류 등의 메시지를 출력하는 메서드를 제공하는 객체
  • navigator -> 브라우저의 정보를 담고있는 객체
    • // 브라우저의 이름, 버전, 플랫폼, 언어 등의 정보를 담고 있다.
